# Tasky Backend

# Technologies

- Express
- Mongoose
- Express-generator
- morgan (for log)

# To Run locally

1. Clone the repo

```
git clone https://github.com/434huzaifa/Tasky-server
```

2. Go the Cloned Directory

```
cd Tasky-server
```

3. Install dependencies

```
npm i
```

4. Run the project

```
npm i nodemon -g
```

```
nodemon
```
